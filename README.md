# Projeto (API) 
O projeto tem como objetivo desenvolver uma plataforma de Business Intelligence voltada à análise da produtividade e eficiência operacional dos terminais portuários brasileiros, utilizando dados públicos da ANTAQ. A proposta consiste na construção de um dashboard web interativo capaz de processar e integrar dados sobre tempos de operação, movimentação de cargas e paradas portuárias, permitindo uma análise consolidada em nível de terminal e nacional. A partir da organização dessas informações, o sistema busca apoiar a identificação de gargalos logísticos, a comparação de desempenho entre terminais e o aprimoramento da eficiência no setor portuário.

# Objetivo do Projeto
Desenvolver um dashboard interativo que integre dados da ANTAQ para analisar a eficiência operacional dos terminais portuários brasileiros. O sistema deve apresentar métricas consolidadas por terminal e em nível nacional, incluindo tempos de operação, volume de cargas movimentadas, quantidade de operações, principais motivos de paradas e tipos de cargas, permitindo análises comparativas e apoiando a tomada de decisão no contexto logístico portuário.
